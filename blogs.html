Question-1: What are the different ways to select an element in the DOM?

By ID (getElementById), Class (getElementsByClassName), Tag (getElementsByTagName), CSS selectors (querySelector/querySelectorAll), and XPath.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML includes HTML tags, innerText shows visible text only, and textContent returns all text content including hidden elements.

Question-3: What is event delegation in the DOM?

Event delegation uses a single parent event listener to manage events for multiple child elements through event bubbling.

Question-4: What is event bubbling in the DOM?

Event bubbling is when events triggered on child elements propagate upward through parent elements in the DOM tree.

Question-5: How do you create, add, and remove elements using JavaScript?

createElement creates elements, appendChild adds them, and removeChild deletes them from the DOM.
